# Brasil Encantado

---

## O Projeto

O Brasil Encantado é um site de uma empresa de turismo ficticia, voltada a mostrar as belezas que o Brasil esconde, tanto geográficas quanto culinárias/culturais.

Desenvolvido para o curso de Ciência da Computação pela Escola de Engenharia de Piracicaba (1° Semestre).
